# Smart Farming AI Platform

AI-powered smart farming platform that helps farmers choose the best crops using weather, soil, and market data, while also detecting crop diseases from plant images.

## Problem Statement
PS-05: Smart Farming Decision Support Platform  
Additional Feature: Crop Disease Detection (PS-04)

## Features
- 🌾 Crop recommendation using weather and soil data
- 📷 AI-based crop disease detection from plant images
- 📊 Crop planning dashboard
- ⚠️ Weather alerts for farmers
- 💡 Smart farming insights

## Tech Stack
- Frontend: React / HTML / CSS
- Backend: Node.js / Python
- AI Model: TensorFlow / PyTorch
- Database: MongoDB / Firebase

## How It Works
1. Farmer enters location and soil information.
2. System analyzes weather and market data.
3. AI recommends the best crops to grow.
4. Farmers can upload plant images for disease detection.
5. Platform provides treatment suggestions.
