# 🌱 AgriVeista – Smart Farming Decision Support Platform

AgriVeista is an intelligent farming assistant designed to help farmers make better agricultural decisions using technology.
The platform provides crop recommendations, farming insights, and smart agricultural guidance based on environmental conditions.

This project was developed for **Swarajya Hack Fest 2026**, a national level hackathon focused on solving real-world problems using technology.

---

# 📌 Problem Statement

### PS-05 – Smart Farming Decision Support Platform

Build a digital system that helps farmers decide what crops to grow using:

* Weather data
* Soil information
* Market insights
* Environmental conditions

### Additional Feature

### PS-04 – Crop Disease Detection

Allow farmers to upload crop images and detect possible diseases using AI.

---

# 🚜 Project Overview

Agriculture is highly dependent on multiple factors such as soil quality, weather conditions, and crop diseases. Many farmers do not have access to proper tools to analyze these factors.

AgriVeista solves this problem by providing a simple digital platform that:

* Helps farmers choose the best crops
* Provides insights about farming conditions
* Assists in early detection of crop diseases
* Improves decision making using technology

The platform aims to make farming **smarter, data-driven, and more efficient**.

---

# 🧠 System Architecture

The platform follows a simple web architecture.

User → Web Interface → Processing Logic → Results Display

1. User enters farming data or uploads crop images
2. The system processes the data
3. Crop recommendation or disease analysis is generated
4. Results are displayed on the dashboard
